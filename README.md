# 🌟 Syanai - Webアプリ ポートフォリオ

このリポジトリは、EclipseとSpring Bootを使用して作成したWebアプリケーションのポートフォリオです。

---

## 📦 プロジェクト一覧

### 📌 3Fpodproject
- Spring BootをベースにしたWebアプリケーション
- 会員登録、ログイン、タスクリストの機能を実装
- HTMLテンプレートは `src/main/resources/templates` にあります

---
